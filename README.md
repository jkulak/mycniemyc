# Myć Czy Nie Myć 🚗💦

**MyćCzyNieMyć** (To Wash or Not to Wash) is a simple, browser application for drivers. It answers the eternal question: *"Should I wash my car today, or is it going to rain soon anyway?"*. 

The app checks the forecast by fetching data via the free [Open-Meteo API](https://open-meteo.com/), and its algorithm comes down to checking if zero precipitation is expected over the coming days (e.g., the next 3 days) – giving the driver a clear prompt: **MYĆ!** (WASH!) or **NIE MYĆ!** (DO NOT WASH!).

> **Twist usage:** You can also use this app for high-porosity hair that gets frizzy from humidity - it's not worth washing and styling if it's going to rain anyway! 💇‍♀️☔

---

## 🚀 How it works
- 100% Client-side. The app does not require **any backend** or API keys.
- Built with HTML, Vanilla CSS and Vanilla JS.
- Optional use of the Geolocation API (with user consent) or a manual location input field (with a fallback search engine).
- Supports Dark Mode and Light Mode.
- Supports Polish and English.
